# Προσωπικός Ιστότοπος – Πανεπιστημιακό Project

Αυτό το αποθετήριο περιέχει τον προσωπικό μου ιστότοπο, που δημιουργήθηκε ως πανεπιστημιακή εργασία. Ο ιστότοπος αποτελείται από 5 σελίδες:

- **Αρχική (Home)**: Κεντρική σελίδα με hero section και χαιρετισμό.
- **Σχετικά (About)**: Σχετικά με εμένα, με περιγραφή και φωτογραφία.
- **Portfolio**: Συλλογή των έργων μου (σε περίπτωση που επιλεγεί).
- **Gallery**: Συλλογή εικόνων/φωτογραφιών (σε περίπτωση που επιλεγεί).
- **Επικοινωνία (Contact)**: Φόρμα επικοινωνίας με έλεγχο υποχρεωτικών πεδίων και περιορισμό χαρακτήρων.

## Τεχνολογίες

- HTML5
- CSS3
- JavaScript (χωρίς βιβλιοθήκες)
- Git & GitHub για διαχείριση εκδόσεων
